# Schedulr Backend API

This repository contains the backend API for the Schedulr application, a comprehensive scheduling tool designed
to streamline and optimize the planning process.
The backend is responsible for handling data processing, storage, and communication
between the Schedulr frontend application and the database, ensuring a seamless and efficient user experience.

For more information about the Schedulr frontend application, visit [Schedulr Frontend](https://github.com/stevendejongnl/schedulr).
